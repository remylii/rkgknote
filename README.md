# rkgknote

[![Build Status](https://travis-ci.org/remylii/rkgknote.svg?branch=master)](https://travis-ci.org/remylii/rkgknote)

## Stack

- package manager: yarn
- transpiler: babel
- linter: eslint, flow
- test runner: ava
- coverage tools: nyc
- builder: webpack
- css: postcss
- minify: csswring, ugfily-js
- server: http-server, browser-sync

## Usage

```
$ yarn install
$ yarn start
#($ yarn start:local)
```

## Tasks

Use npm-scripts.

- build:js
- build:css
- watch:js
- watch:css
- compress:js
- compress:css
- typecheck
- lint
- test
- cover
